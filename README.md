# lab2-network
Лабораторна робота №2
Проєктування локальної мережі офісу
 Опис приміщення
Було обрано офісне приміщення:
- ліва кімната 5×2 м
- права кімната 5×2 м
- центральна серверна 2×1,5 м

 Обладнання
- роутер
- комутатор
- NAS
- сервер
- 4 ПК
- 2 ноутбуки
- принтер
- Smart TV
- мережеві розетки
- кабель UTP Cat5e

Схема
Тут буде вставлена схема з draw.io

Кошторис
| Найменування | Ціна | Кількість | Вартість |
|---|---:|---:|---:|
| Роутер TP-Link Archer C6 | 2200 | 1 | 2200 |
| Комутатор 8-port | 1400 | 1 | 1400 |
| Кабель UTP Cat5e | 12 | 20 | 240 |
| Конектор RJ-45 | 2 | 8 | 16 |
| Мережева розетка | 120 | 4 | 480 |
| NAS | 5000 | 1 | 5000 |
| Сервер | 8000 | 1 | 8000 |
| Принтер | 4500 | 1 | 4500 |
| РАЗОМ |  |  | 21836 |

 Висновок
Було спроєктовано локальну мережу офісного приміщення, підібрано сумісне обладнання та розраховано його вартість.# Лабораторна робота №2
