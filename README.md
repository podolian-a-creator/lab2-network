 Лабораторна робота №2
 Проєктування локальної мережі офісного приміщення

 Опис приміщення
Було обрано офісне приміщення, що складається з:
- робочий кабінет 1 (5×2 м)
- робочий кабінет 2 (5×2 м)
- серверна (2×1,5 м)

Склад обладнання
- роутер
- комутатор 8-port
- NAS
- сервер
- 4 ПК
- 2 ноутбуки
- мережевий принтер
- Smart TV
- мережеві розетки
- кабель UTP Cat5e

## Розрахунок довжини кабелю
Загальна довжина кабелю UTP Cat5e була розрахована як сума відстаней від комутатора до всіх пристроїв, підключених через Ethernet, з урахуванням прокладання по стінах, висоти стелі 3 м та запасу на монтаж.

**Загальна довжина кабелю: 42 м**
